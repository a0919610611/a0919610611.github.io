# 為什麼會買 Mac 呢
大家常常在說「Mac 是盤子在買，拿去星巴克耍潮而已」，不然就是「還是拿去灌 Windows」，這是許多人對 Mac 的「刻板印象」。
身為一個資工系的學生，當然不會去做拿 Mac 去灌 Windows這種奇怪的事。
